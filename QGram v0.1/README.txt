QGram — клиент Telegram, с которым вы можете авторизоваться как бот! Всё происходит в браузере и полностью безопасно.

ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ:

1. Установите Python, перейдите в CMD, затем скопируйте путь к папке и вставьте команду cd your/path/to/folder в CMD.

2. Введите pip install -r requirements.txt в CMD, после чего начнётся загрузка модулей.

3. Создайте своего бота в Telegram и скопируйте токен.

4. Запустите файл main.py из папки и вставьте токен который в скопировали

5. Пользуйтесь!


Программа находится на стадии бета-тестирования
